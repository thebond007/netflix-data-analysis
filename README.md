# Netflix Data Analysis

## Project Overview
This project analyzes Netflix Movies and TV Shows using Python, Pandas, and Matplotlib.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Movies vs TV Shows comparison
- Content rating distribution
- Movie duration analysis
- Release year trends
- Top countries by content
- Movies vs TV Shows trend comparison

## Key Insights
- Movies outnumber TV Shows on Netflix
- Netflix content increased over the years
- Some countries contribute significantly more content
- Movie duration follows common patterns

## Dataset
Netflix Titles Dataset
